# 📦 AI Object Detection System (YOLOv8)

This project is part of CodeAlpha AI Internship Task 04.  
It detects real-world objects using webcam in real time.

---

## ✨ Features

- Real-time object detection  
- YOLOv8 AI model  
- Webcam live feed  
- Bounding boxes with labels  
- Confidence score display  

---

## 🧠 Technologies Used

- Python  
- Streamlit  
- OpenCV  
- YOLOv8 (Ultralytics)  

---

## Installation

```bash
pip install -r requirements.txt
```

## Run Project

```bash
streamlit run app.py
```
