# 🎓 AI University Admission Assistant (FAQ Chatbot)

A modern AI-powered FAQ Chatbot developed as part of the CodeAlpha Artificial Intelligence Internship. This project helps students get instant answers related to university admissions, scholarships, fees, hostel, programs, and other academic queries using NLP techniques (TF-IDF + Cosine Similarity).

## Features

- Smart FAQ-based AI chatbot  
- NLP-based text processing  
- TF-IDF + Cosine Similarity matching  
- ChatGPT-style Streamlit UI  
- Confidence score for each answer  
- Chat history support  
- Sidebar with quick questions  
- Clean and modern interface  

## Technologies Used

- Python  
- Streamlit  
- Pandas  
- NLTK (Natural Language Processing)  
- Scikit-learn (TF-IDF, Cosine Similarity)  

## Installation

```bash
pip install -r requirements.txt
```

## Run Project

```bash
streamlit run app.py
```
