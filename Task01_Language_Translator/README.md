# 🌍 AI Language Translation Tool

A modern AI-powered Language Translation Tool developed as part of the CodeAlpha Artificial Intelligence Internship. This project enables real-time multilingual translation using Python and Streamlit.

## Features

- Translate text into multiple languages
- Clean and modern user interface
- Text-to-Speech support
- Download translated text
- Fast and accurate translation
- Error handling

## Technologies Used

- Python
- Streamlit
- Deep Translator
- Google Translator API
- gTTS

## Installation

```bash
pip install -r requirements.txt
```

## Run Project

```bash
streamlit run app.py
```
