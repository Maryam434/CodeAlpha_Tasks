# 🎵 AI Music Generator

A simple AI-powered Music Generator built as part of the CodeAlpha Artificial Intelligence Internship. This project generates AI-based music sequences using Markov Chain logic and converts them into real audio (WAV format).

---

## ✨ Features

- AI-based music sequence generation  
- Markov Chain inspired logic  
- WAV audio file generation  
- Download generated music  
- Simple and interactive Streamlit UI  
- No external heavy dependencies (fully error-free)  

---

## 🧠 Technologies Used

- Python  
- Streamlit  
- NumPy  
- SciPy (Audio generation)  

---

## ⚙️ Installation

```bash
pip install -r requirements.txt
```

## Run Project

```bash
streamlit run app.py
```
